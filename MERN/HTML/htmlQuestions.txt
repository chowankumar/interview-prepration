Q1 : what is html?
HTML is a markup language that uses tags to define elements such as headings, paragraphs, links, images, and forms, allowing web browsers to display content correctly on a webpage.
HyperText: Text that contains links to other text or pages
Markup: Uses tags to mark and structure content
Language: A set of rules understood by web browsers

Q2 : What is the function of the browser? 
To read HTML documents and display them.

Q3 : What is a Tag? 
A keyword surrounded by angle brackets (e.g., <html>).

Q4 : What is an Element? 
The start tag, content, and end tag (e.g., <p>Hello</p>).

Q5 : What is an Attribute? Provides additional info about an element.

Q6 : What is the root element of an HTML page? 
<html>.

Q7 : What is the <!DOCTYPE>? 
It's not a tag; it's a declaration telling the browser the HTML version (HTML5).

Q8 : What happens if you remove <!DOCTYPE>? 
The browser goes into "Quirks Mode" and may render layout incorrectly.

Q9 : Is HTML case sensitive? 
No, but lowercase is the standard convention (and required in JSX).

Q10 : Where do you define metadata? 
In the <head> section.

Q11 : Where does the visible content go? 
In the <body> section.

Q12 : How do you add comments in HTML? 
<!-- Comment here -->.

Q13 : Can comments be seen in the browser? 
Not on the UI, but yes in the "View Source" code

Q14 : What is an Empty (Void) Element? 
An element with no closing tag (e.g., <br>, <img>, <input>).

Q15 : What is nesting? 
Placing elements inside other elements.

Q16 : What is the correct extension for HTML files? 
.html or .htm.

Q17 : What is the lang attribute in the <html> tag? 
It declares the language (e.g., en) for screen readers and search engines.

Q18 : What character encoding is recommended? 
UTF-8. is liye recommend ki jati hai kyun ke yeh almost tamam zabanon ke haroof aur symbols ko support karti hai.

Q19 : How do you set the character encoding? 
<meta charset="UTF-8">.

Q20 : What is the title tag used for? 
The text shown on the browser tab and search engine results.

Q21 : What is a Block-level element?
A block-level element starts on a new line and takes the full width of the page (e.g., <div>, <p>, <h1>).

Q22 : What is an Inline element?
An inline element stays on the same line and only takes as much width as needed (e.g., <span>, <a>, <b>).

Q23 : What is <div>?
<div> is a generic block-level container used to group elements for layout or styling.

Q24 : What is <span>?
<span> is a generic inline container used to style or modify small parts of text.

Q25 : How many heading tags are there?
There are six heading tags in HTML: <h1> to <h6>.

Q26 : Which heading is most important for SEO?
<h1> is the most important because it defines the main topic of the page.

Q27 :  Should you use headings just to resize text?
No, headings are for structure; text resizing should be done using CSS.

Q28. What does <p> stand for?
<p> stands for paragraph and is used to write blocks of text.

Q29. What does <br> do?
<br> inserts a single line break without starting a new paragraph.

Q30. What does <hr> do?
<hr> creates a horizontal line to show a thematic break between content.

Q31. What is <pre>?
<pre> shows preformatted text and keeps spaces and line breaks as written.

Q32. Difference between <b> and <strong>?
<b> makes text bold visually, while <strong> shows importance and is meaningful for screen readers.

Q33. Difference between <i> and <em>?
<i> italicizes text visually, while <em> adds emphasis with semantic meaning.

Q34. What is <small>?
<small> is used for side notes, disclaimers, or copyright text.

Q35. What is <mark>?
<mark> highlights text, usually with a yellow background.

Q36. What is <del>?
<del> shows deleted or removed text with a line through it.

Q37. What is <ins>?
<ins> shows inserted or added text, usually underlined.

Q38. What is <sub>?
<sub> displays subscript text, like H₂O.

Q39. What is <sup>?
<sup> displays superscript text, like X².

Q40. What is <blockquote>?
<blockquote> is used for long quotations taken from another source.

Q41. What is <q>?
<q> is used for short inline quotations and adds quotation marks automatically.

Q42. What is <abbr>?
<abbr> defines abbreviations and shows full meaning on hover using the title attribute.

Q43. What is <address>?
<address> contains contact information of the author or website owner.

Q44. What is <code>?
<code> is used to display programming or computer code.

Q45. What is <kbd>?
<kbd> represents keyboard input, such as Ctrl + C.

46. How do you make an unordered list?
An unordered list is created using the <ul> tag. It displays items with bullets (dots) by default and is used when the order of items does not matter.

47. How do you make an ordered list?
An ordered list is created using the <ol> tag. It displays items in a specific order using numbers or letters.

48. What creates a list item?
The <li> tag creates a list item. It is used inside <ul> or <ol> to define each item in the list.

49. How do you change the bullet style of <ul>?
You change the bullet style using CSS list-style-type (e.g., disc, circle, square). The old HTML type attribute is deprecated and not recommended.

50. How do you change the numbering of <ol> to letters?
You can change numbers to letters by using the type attribute, for example: <ol type="A"> for uppercase letters or <ol type="a"> for lowercase.

51. How do you start an <ol> from number 5?
Use the start attribute like <ol start="5">. This makes the first list item begin with number 5.

52. What is a Description List?
A Description List is created using the <dl> tag. It is used to display terms and their descriptions, like a dictionary or glossary.

53. What is a Description Term?
The <dt> tag defines a term or name in a description list.

54. What is a Description Data/Definition?
The <dd> tag defines the description or explanation of the term written in <dt>.

55. Can you nest lists?
Yes, lists can be nested. For example, a <ul> or <ol> can be placed inside an <li> to create sub-lists.

56. What tag defines a table?
The <table> tag is used to create a table structure in HTML.

57. What is a table row?
A table row is defined using the <tr> tag. It contains table header or data cells.

58. What is a table header cell?
The <th> tag defines a table header cell. It is bold and center-aligned by default.

59. What is a table data cell?
The <td> tag defines a normal data cell in a table.

60. How do you merge columns?
Columns are merged using the colspan attribute, for example: colspan="2".

61. How do you merge rows?
Rows are merged using the rowspan attribute, for example: rowspan="2".

62. What is <thead>?
<thead> groups the header content of a table, usually containing column headings.

63. What is <tbody>?
<tbody> groups the main body content of a table where the actual data rows are placed.

64. What is <tfoot>?
<tfoot> groups the footer content of a table, often used for totals or summary rows.

65. What is <caption> in a table?
<caption> provides a title or heading for the table, helping users understand its purpose.

66. Why use scope in <th>?
The scope attribute (like scope="col" or scope="row") helps screen readers understand which data cells relate to which headers, improving accessibility.

67. Is it okay to use tables for page layout?
No, tables should never be used for page layout. They are meant only for tabular data. For layout design, use CSS Flexbox or Grid.

68. What tag is used for links?
The <a> (anchor) tag is used to create hyperlinks. It allows users to navigate from one page to another or to a different section of the same page.

69. What attribute holds the URL?
The href attribute holds the URL or destination address where the link points.

70. What happens if href is missing?
If the href attribute is missing, the anchor tag does not act as a clickable link and behaves like plain text or a placeholder.

71. What is an Absolute URL?
An absolute URL contains the complete web address, including the protocol, domain, and path (e.g., https://google.com).

72. What is a Relative URL?
A relative URL specifies a path relative to the current file or domain (e.g., /images/logo.png or about.html).

73. How do you open a link in a new tab?
You use the attribute target="_blank" to open the link in a new browser tab or window.

74. What is the security risk of target="_blank"?
It can lead to phishing attacks called tabnabbing, where the newly opened page can manipulate the original page.

75. How do you fix the security risk?
By adding rel="noopener noreferrer" to the link, which prevents the new page from accessing the original page.

76. How do you link to an email?
Use href="mailto:abc@example.com" to open the user’s default email client with the address filled in.

77. How do you link to a phone number?
Use href="tel:+123456789" to allow users to directly call the number on supported devices.

78. What is a Fragment Identifier?
A fragment identifier links to a specific section on the same page using an ID (e.g., href="#section1").

79. What is the download attribute?
The download attribute forces the browser to download the file instead of opening or navigating to it.

80. What color is an unvisited link by default?
By default, an unvisited link appears blue in most browsers.

81. What color is a visited link by default?
By default, a visited link appears purple, indicating it has already been clicked.

82. What tag is used for images?
The <img> tag is used to display images on a web page. It is a self-closing (void) tag and does not have a closing tag.

83. Is <img> a block or inline element?
<img> is an inline-block element (technically a replaced element). It flows inline with text but can have width and height like a block.

84. Why is alt text required?
Alt text is required for accessibility so screen readers can describe the image to visually impaired users. It also helps with SEO and displays if the image cannot load.

85. What happens if src is broken?
If the image source is broken or missing, the browser displays the alt text instead of the image.

86. What is the title attribute on images?
The title attribute shows a small tooltip when the user hovers over the image, giving extra information.

87. Why should you specify width and height attributes?
Specifying width and height helps prevent Cumulative Layout Shift (CLS) by reserving space for the image before it loads.

88. What is <figure>?
<figure> is a semantic HTML tag used to wrap images, diagrams, or media that are referenced in the content.

89. What is <figcaption>?
<figcaption> provides a caption or description for the content inside <figure>.

90. What file formats support transparency?
PNG, WebP, GIF, and SVG support transparency. JPEG does not support transparent backgrounds.

91. What is SVG?
SVG (Scalable Vector Graphics) is a vector image format based on XML. It can scale to any size without losing quality or becoming blurry.

92. What is the <map> tag?
The <map> tag defines an image map, allowing different parts of an image to be clickable.

93. What is <area>?
The <area> tag defines clickable regions inside an image map, such as rectangles, circles, or polygons.

94. How do you embed audio?
Audio is embedded using the <audio> tag with the src attribute and controls to show play options.

95. How do you embed video?
Video is embedded using the <video> tag with the src attribute and controls.

96. What does the autoplay attribute do?
The autoplay attribute makes media play automatically, but most browsers block it if the media has sound.

97. What does the loop attribute do?
The loop attribute causes the audio or video to restart automatically when it ends.

98. What does poster do in video?
The poster attribute displays an image before the video starts playing.

99. What is the <source> tag?
The <source> tag is used inside <audio> or <video> to provide multiple file formats for better browser support.

100. What is the <track> tag?
The <track> tag is used to add subtitles, captions, or other text tracks to videos, improving accessibility.

101. What tag defines a form?
The <form> tag is used to create an HTML form that collects user input and sends it to a server.

102. What is the action attribute?
The action attribute specifies the URL where the form data is sent after submission. In React/MERN apps, this is often handled by JavaScript instead.

103. What is the method attribute?
The method attribute defines the HTTP method used to send form data, usually GET or POST.

104. What is <input>?
The <input> tag is the most versatile form element, used for many types of user input.

105. List 5 common input types.
Common input types include text, password, checkbox, radio, and submit.

106. What is type="password"?
It hides the entered characters using dots or asterisks for security.

107. What is type="email"?
It automatically validates that the entered value follows a proper email format.

108. What is type="number"?
It allows only numeric input and usually provides up/down spinner controls.

109. What is type="date"?
It opens a built-in date picker so users can easily select a date.

110. What is type="checkbox"?
Checkbox inputs allow users to select multiple options.

111. What is type="radio"?
Radio buttons allow users to select only one option from a group.

112. How do you group radio buttons?
Radio buttons are grouped by giving them the same name attribute.

113. What is type="file"?
It allows users to upload files from their device.

114. What is type="hidden"?
It stores data that is not visible to the user but is sent to the server.

115. What is type="submit"?
It submits the form data to the server or triggers form handling logic.

116. What is type="reset"?
It resets all form fields to their default values.

117. What is type="color"?
It opens a color picker so users can choose a color visually.

118. What is type="range"?
It provides a slider control to select a value within a range.

119. What is the <textarea> tag?
<textarea> is used for multi-line text input, such as comments or messages.

120. Is <textarea> self-closing?
No, it requires opening and closing tags: <textarea>Text</textarea>.

121. What is <select>?
The <select> tag creates a dropdown list.

122. What is <option>?
<option> defines each selectable item inside a <select> dropdown.

123. What is <optgroup>?
It groups related options within a dropdown list and provides a label.

124. What is <label>?
The <label> tag provides captions for form controls and improves accessibility.

125. How do you bind a label to an input?
By using for="input_id" (in React, use htmlFor instead of for).

126. What is <fieldset>?
<fieldset> groups related form elements together and displays them in a box.

127. What is <legend>?
<legend> provides a caption or title for a <fieldset>.

128. What is <datalist>?
It provides predefined autocomplete suggestions for an input field.

129. What is the placeholder attribute?
It shows hint text inside an input field to guide the user.

130. What is the value attribute?
It defines the initial or current value of an input field.

131. What is readonly?
The user can see the value but cannot modify it, and it is sent to the server.

132. What is disabled?
The input cannot be used, and its value is not sent to the server.

133. What is required?
The form cannot be submitted unless the field is filled.

134. What is autofocus?
It automatically focuses on the input when the page loads.

135. What is maxlength?
It limits the maximum number of characters a user can enter.

136. What is min and max?
They define the minimum and maximum allowed values for number or date inputs.

137. What is step?
It specifies valid number intervals, such as step="10".

138. What is multiple?
It allows selecting multiple files or multiple options in a dropdown.

139. What is pattern?
It uses a regular expression to validate input format (e.g., [A-Za-z]{3}).

140. What is novalidate on a form?
It disables the browser’s default form validation, allowing custom validation logic

141. What are Semantic Elements?
Semantic elements are HTML tags that clearly describe their meaning and purpose (e.g., <form>, <table>, <article>). Non-semantic elements like <div> and <span> do not convey any meaning about their content.

142. What is <header>?
<header> represents the introductory content of a page or section. It often contains headings, logos, or navigation links.

143. What is <nav>?
<nav> is used to define a section that contains navigation links, such as menus or breadcrumbs.

144. What is <section>?
<section> groups related content under a common theme. It usually includes a heading and organizes the page logically.
<section>
  <h2>Services</h2>
  <p>Web development</p>
</section>

145. What is <article>?
<article> represents independent, self-contained content such as a blog post, news article, or forum post.independent content hota hai jo akela bhi use ho sakta hai, jaise blog ya news.
<article>
  <h2>Blog Title</h2>
  <p>Blog content</p>
</article>


146. What is <aside>?
<aside> contains content indirectly related to the main content, like sidebars, ads, or related links.
<aside>
  <p>Related links</p>
</aside>

147. What is <footer>?
<footer> defines the footer area of a page or section and typically contains copyright information, contact details, or legal links.

148. What is <main>?
<main> holds the unique, central content of the document. There must be only one <main> per page, and it should not be nested.

149. What is <details>?
<details> creates a disclosure widget that allows users to expand or collapse additional content.
<details>
  <summary>More info</summary>
  <p>Hidden text</p>
</details>

150. What is <summary>?
<summary> provides the visible heading or label for a <details> element that users click to toggle content.

151. What is <time>?
<time> represents a specific date or time in a machine-readable format, useful for SEO and accessibility.

152. What is <dialog>?
<dialog> defines a native HTML popup or modal window, such as alerts or confirmation boxes.

153. Why use semantic tags?
Semantic tags improve SEO, accessibility for screen readers, and code readability/maintainability.

154. Can you have multiple <header> tags?
Yes, multiple <header> tags are allowed—one for each section or article.

155. Can you have multiple <h1> tags?
Technically yes in HTML5, but most SEO experts recommend using one <h1> per page for best SEO practices.

156. What is the Viewport Meta tag?
It controls how a web page is displayed on different screen sizes, especially mobile devices, by defining the visible area (viewport).

157. What is the syntax for the viewport meta tag?
<meta name="viewport" content="width=device-width, initial-scale=1.0">
This sets the page width to the device’s width and the default zoom level to 1.

158. What is meta name="description"?
It provides a short summary of the page content, often shown under the page title in Google search results.

159. What is meta name="keywords"?
An old SEO tag used to list keywords; it is now obsolete and ignored by modern search engines like Google.

160. What is meta name="robots"?
It tells search engines how to crawl and index a page (e.g., index, noindex, follow, nofollow).

161. What is noindex?
A directive that tells search engines not to include the page in search results.

162. What are Open Graph (OG) tags?
Meta tags used by social media platforms (like Facebook and LinkedIn) to control how links are previewed when shared.

163. What is a Favicon?
A small icon that represents a website, shown in the browser tab, bookmarks, and address bar.

164. How do you link a Favicon?
<link rel="icon" href="icon.png">
This connects the favicon image to the webpage.

165. What is a Canonical tag?
<link rel="canonical" href="URL">
It tells search engines the preferred version of a page to avoid duplicate content SEO issues.

166. What does A11y stand for?
A11y ka matlab Accessibility hai (A aur y ke beech 11 letters). Ye ensure karta hai ke website har user, including disabled users, easily use kar saken.
Example: Screen readers ke liye properly labelled buttons.

167. What is ARIA?
ARIA ka matlab Accessible Rich Internet Applications hai. Ye dynamic web content ko screen readers aur assistive tech ke liye accessible banata hai.
<div role="button">Click me</div>

168. What is aria-label?
aria-label invisible elements ko text label provide karta hai, jisse screen readers samajh saken.
<button aria-label="Close menu"><i class="icon-close"></i></button>

169. What is role?
role batata hai element ka purpose ya type, jaise button, navigation, etc., taake assistive devices samajh saken.
<div role="button">Click me</div>

170. What is tabindex="0"?
tabindex="0" kisi non-interactive element ko keyboard se focusable banata hai, aur natural tab order me include karta hai.
<div tabindex="0">Focusable Div</div>

171. What is tabindex="-1"?
tabindex="-1" element ko tab se skip karta hai; sirf JavaScript ke through focus kiya ja sakta hai.
<div tabindex="-1">Not focusable by Tab</div>

172. Why is contrast important?
High contrast text aur background visually impaired users ke liye zaroori hai, taake text easily read ho.
Example: White text on dark background.

173. What is a "Skip to Content" link?
Ye hidden link keyboard users ko navigation menus skip karke directly main content par le jata hai.
<a href="#maincontent" class="skip-link">Skip to Content</a>
